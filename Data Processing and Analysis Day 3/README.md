# Data Processing and Analysis - Day 3 Practice

This repository contains my **Day 3 Practice** notebook focused on **Python Data Processing and Analysis** using basic programming concepts such as loops, conditions, dictionaries, lists, and data validation.

### 1. Employee Performance Bonus Eligibility
- Store employee scores using a dictionary  
- Find highest score  
- Identify top performers eligible for bonus  

### 2. Search Query Keyword Analysis
- Convert text to lowercase  
- Split query into words  
- Count repeated keywords using dictionary  

### 3. Sensor Data Validation
- Process sensor readings list  
- Detect valid even-number readings  
- Display hour and value  

### 4. Email Domain Usage Analysis
- Extract email domains  
- Count users per domain  
- Calculate percentage usage  

### 5. Sales Spike Detection
- Calculate average sales  
- Detect spikes above threshold  
- Print high-sales days  

### 6. Duplicate User ID Detection
- Count repeated user IDs  
- Display duplicate entries with frequency
