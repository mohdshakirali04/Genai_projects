# Logic Building Day 2 🧠🐍

This repository contains my **Day 2 Python Logic Building Practice** notebook.  
It includes beginner-friendly Python problems focused on improving problem-solving skills, loops, conditions, dictionaries, lists, sets, tuples, and basic calculations.

## 📁 File Included

- `Logic Building Day 2.ipynb` – Jupyter Notebook containing 10 logic-building Python programs.

---

## 🚀 Topics Covered

### ✅ Problems Solved

1. **Unique Words in a Sentence**  
   Count and display unique words using `set()`.

2. **Highest Salary from Employee Data**  
   Find the employee with the highest salary using dictionary traversal.

3. **Find Maximum and Minimum Values**  
   Find largest and smallest numbers from a list.

4. **Count Products Above a Price Threshold**  
   Count products with price greater than a given value.

5. **Calculate Attendance Percentage**  
   Calculate attendance percentage from present/absent records.

6. **Remove Duplicate Phone Numbers**  
   Remove duplicate values using `set()`.

7. **Count Character Frequency**  
   Count frequency of each character in a string using dictionary.

8. **Convert List to Tuple**  
   Convert list data into tuple format.

9. **Check if a Key Exists in Dictionary**  
   Verify whether a key exists in dictionary or not.

10. **Calculate Average Marks**  
   Find average marks from a list.
